# Capital Markets Trading Analytics

## Overview
An end-to-end Capital Markets Analytics project built using Python, MySQL, SQL, and Power BI.

## Tech Stack
- Python
- MySQL
- SQL
- Pandas
- NumPy
- Power BI

## Features
- Synthetic financial data generation
- Data cleaning and validation
- Feature engineering
- ETL pipeline
- MySQL database
- SQL analytics
- Interactive Power BI dashboard (coming soon)
